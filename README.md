# 🍅 超时专注 (Overtime Focus)

一款基于 Electron 打造的极简桌面专注工具。**与传统番茄钟最大的不同在于：我们拥有独特的“超时记录”功能，保护你来之不易的心流状态。**

## ✨ 核心亮点：为什么你需要“超时记录”？

传统番茄钟在倒计时结束时往往会用刺耳的闹钟强行打断你，而 Smart Tomato 选择了顺应你的专注节奏：

- 🔥 **无缝进入超时模式**：倒计时归零后不强制中断，自动无缝切换为正向计时，精准记录你额外投入的专注时间。
- 📈 **心流数据可视化**：历史记录中会清晰拆分并标记“计划时间”与“超时时间”，让你看见自己超越极限的每一分钟。
- 🪟 **极简悬浮窗联动**：支持一键切换为桌面小浮窗。无论是正常倒计时还是超时标红警告，双端状态实时同步，不遮挡工作视线。

## 📥 快速开始

**普通用户下载体验：**
前往本仓库的 [Releases](https://github.com/abiharomatu77-a11y/SmartTomato/releases) 页面，下载最新的版本，双击即可运行。

**开发者本地运行：**

```bash
git clone [https://github.com/abiharomatu77-a11y/SmartTomato.git](https://github.com/abiharomatu77-a11y/SmartTomato.git)
cd SmartTomato
npm install
npm start
```
